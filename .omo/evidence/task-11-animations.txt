How to use the new API
----------------------
- Existing pages:  no change. Legacy [data-animate] still reveals.
- New opt-in reveal:  add class="fade-in" (or slide-in-left / slide-in-right / scale-in)
  The element stays hidden until it scrolls into view, then fades in.
- Custom reveal type:  <h2 data-animation="slide-in-left">...</h2>
- Custom delay:  <div data-animation="fade-in" data-delay="300">...</div>
- Stagger group:  <ul data-stagger-children>
                    <li class="fade-in">...</li>
                    <li class="fade-in">...</li>
                    <li class="fade-in">...</li>
                  </ul>
  Children get .stagger-1, .stagger-2, .stagger-3 automatically.
- No code change needed to enable hover micro-interactions on cards / CTAs / nav links.
  main.js wires them automatically on DOMContentLoaded.
