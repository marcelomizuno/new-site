## Module summary

### `js/main.js`
- Uses IIFE + `DOMContentLoaded`.
- Toggles `.hamburger.active`, `aria-expanded`, and `.mobile-nav.open` visibility.
- Intercepts `a[href^="#"]` links and calls `scrollIntoView({ behavior: 'smooth', block: 'start' })`.
- Adds/removes `header.scrolled` when `window.scrollY > 50`.
- Observes `[data-animate]` with Intersection Observer and adds `.animate-in`.
- Ensures `.whatsapp-float` is fixed bottom-right.

### `js/faq.js`
- Uses IIFE + `DOMContentLoaded`.
- Selects all `.faq-item` elements.
- Toggles `.active` on the parent FAQ item from `.faq-question` clicks.
- Sets `.faq-answer` `maxHeight` to `scrollHeight + 'px'` when active and `0px` when inactive.
- Closes other FAQ items so only one answer is open at a time.

### `js/form.js`
- Uses IIFE + `DOMContentLoaded`.
- Finds all `form[data-validate]`.
- Prevents native submit and performs client-side validation only.
- Validates required fields and email fields by requiring `@` and `.`.
- Shows `.error` text next to invalid fields.
- Clears errors on valid input/blur.
- Shows `.success-message` and hides the submitted form after valid submission.

### `js/language.js`
- Uses IIFE + `DOMContentLoaded`.
- Defines translations for `en`, `pt`, and `es`.
- Includes CTA text, nav items, and hero headline/subtitle/eyebrow translations.
- Updates all `[data-i18n]` elements on `.lang-btn` click.
- Stores selected language with `localStorage.setItem('lang', code)`.
- Reads `localStorage.getItem('lang')` on page load and applies it.
- Adds `.active` and `aria-pressed` state to the selected language button.
